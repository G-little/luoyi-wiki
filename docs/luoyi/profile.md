#### 1.0 学校列表

##### 接口说明

根据关键字查询学校列表

##### 请求说明

| http 请求方式          |get             |
|:------------- |:---------------:|
| url      |/profile/school/list |

#####  输入参数

| 参数          |必选             | 类型       | 参数说明        | 备注          |
|:-------------|:---------------:|:-------------|:-------------|:-------------|
| keyword      | 否| string  |  关键字 |   |
| page      | 否 | int  |  分页 |   |
| limit      | 否| int  |  单页条数 |  默认10 |


#####  错误说明





#####  返回实例
```json
{
    "c": 0,
    "m": null,
    "d": {
        "pageSize": 10,
        "total": 5,
        "currentPage": 1,
        "list": [
            {
                "id": 13373,
                "name": "台湾清华大学"
            }
        ],
        "end": true,
        "empty": false,
        "startIndex": 0,
        "totalPage": 1
    }
}

```
