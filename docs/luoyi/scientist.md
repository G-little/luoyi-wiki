


#### 1.1 提交认证

##### 接口说明

提交科学家认证信息

##### 请求说明

| http 请求方式          |post             |
|:------------- |:---------------:|
| url      |/scientist/cert/commit |

#####  输入参数

| 参数          |必选             | 类型       | 参数说明        | 备注          |
|:-------------|:---------------:|:-------------|:-------------|:-------------|
| imageUrl      | 是 | string  |  认证信息 |    |


#####  错误说明





#####  返回实例
```json
{
    "c": 0,
    "m": null,
    "d": {
       
    }
}

```




#### 1.2 获取认证信息

##### 接口说明

获取认证信息

##### 请求说明

| http 请求方式          |get             |
|:------------- |:---------------:|
| url      |/scientist/cert/get |

#####  输入参数


#####  错误说明





#####  返回实例
```json
{
    "c": 0,
    "m": null,
    "d": {
        "uid":1111,
        "certStatus":0, //认证状态 0 初始 1 待审核 2 通过 -1 驳回
        "imageUrl":"/xxxx" //认证图
   }
}

```
