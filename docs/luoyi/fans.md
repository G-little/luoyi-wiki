#### 1.1 关注

##### 接口说明

创建会议

##### 请求说明

| http 请求方式          |put             |
|:------------- |:---------------:|
| url      |/fans/follow |

#####  输入参数

| 参数          |必选             | 类型       | 参数说明        | 备注          |
|:-------------|:---------------:|:-------------|:-------------|:-------------|
| uid      | 是| int  |  用户ID |    |


#####  错误说明





#####  返回实例
```json

{
    "c": 0,
    "m": null,
    "d": {
        "followStatus":0  //关注状态   0:未关注  1：关注  2:双向关注
    }
}

```
