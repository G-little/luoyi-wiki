1.5 用户列表

接口说明

创建会议

请求说明
http 请求方式 get
url /article/user_list
输入参数
参数 必选 类型 参数说明 备注
id 是 int 文章ID
type 否 int 类型 0 浏览用户 1 好奇用户
page 否 int 分页 默认值1
limit 否 int 每页条数 1
