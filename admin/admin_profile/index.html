1.0 学校列表

接口说明

根据关键字查询学校列表

请求说明
http 请求方式 get
url /admin/profile/school/list
输入参数
参数 必选 类型 参数说明 备注
keyword 否 string 关键字
page 否 int 分页
limit 否 int 单页条数 默认10
错误说明
返回实例
{
    "c": 0,
    "m": null,
    "d": {
        "pageSize": 10,
        "total": 5,
        "currentPage": 1,
        "list": [
            {
                "id": 13373,
                "name": "台湾清华大学"
            }
        ],
        "end": true,
        "empty": false,
        "startIndex": 0,
        "totalPage": 1
    }
}

1.2 学科/细分领域 列表

接口说明

根据关键字查询企业列表

请求说明
http 请求方式 get
url /admin/profile/subject/list
输入参数
参数 必选 类型 参数说明 备注
level 否 int 等级 1 学科 2 细分领域
parentId 否 int 父级ID
keyword 否 string 关键字
page 否 int 分页
limit 否 int 单页条数 默认10
错误说明
返回实例
{
    "c": 0,
    "m": null,
    "d": {
        "pageSize": 10,
        "total": 188,
        "currentPage": 1,
        "list": [
            {
                "id": 197,
                "nameCn": "物理",
                "nameEn": "Physics",
                "level": 2,
                "parentId": "31"
            }
        ],
        "end": false,
        "empty": false,
        "startIndex": 0,
        "totalPage": 19
    }
}
