列表接口

请求说明
http 请求方式 get
url /admin/tags/list
输入参数
参数 必选 类型 参数说明 备注
type 否 int 标签类型 1 学科标签 2 细分领域标签 3 其他
page 否 int 分页
limit 否 int 单页条数
返回实例

{
    "c": 0,
    "m": null,
    "d": {
        "pageSize": 10,
        "total": 195,

                "id": 197,

    }
}
